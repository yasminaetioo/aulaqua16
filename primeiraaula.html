Adobe DreamWeaver

Explique o que é e o que podemos fazer com o DreamWeaver

O Dreamweaver é um programa de design online que permite criar, editar e administrar sites e aplicações online.

Podemos desenvolver muitas coisas dentro desse programa, como: Desenhar, codificar e desenvolver sites Modificar um banco de dados usando comandos ASP Criar e projetar aplicativos baseados na Web Desenvolver as habilidades necessárias para o mercado de trabalho Publicar sites em um servidor externo Realizar pequenas alterações rapidamente e vê-las refletidas imediatamente no servidor

Como trabalhos ou fazemos.

Instalação e Configuração: Baixe, instale e configure o Dreamweaver no site da Adobe. Criando um Novo Projeto: Selecione "Arquivo" > "Novo" para criar documentos HTML, CSS, JavaScript, etc. Ambiente de Trabalho: Use as visualizações de "Código" e "Design" para editar e pré-visualizar seu site. Inserindo Elementos: Adicione texto, imagens e links via visualização de "Design" ou escrevendo código. Estilização com CSS: Crie e aplique estilos CSS aos seus documentos. Gerenciamento de Arquivos: Organize arquivos e pastas do seu projeto no painel "Arquivos".

Explique o HTML.

HTML (HyperText Markup Language) é uma linguagem que define a estrutura e o significado do conteúdo de uma página web. É a base da web e é usado para criar websites. O HTML é composto por vários elementos, chamados tags, que marcam diferentes partes do conteúdo. Exemplo, o JavaScript é usado para definir a funcionalidade.

O que são as TAGs, descreva exemplos.

Tags são etiquetas que podem ser usadas para orientar, classificar, estruturar e organizar informações. Explique o CSS.

CSS (Cascading Style Sheets) é uma linguagem de que define como os elementos de uma página da web são apresentados. Basicamente monitora a aparência de um site, como as cores, fontes, espaçamentos, alinhamentos, estilizar, formatar, tamanhos e layouts. O que o CSS faz? Define a aparência de um documento HTML Seleciona elementos da página Controla a apresentação dos elementos da página Permite adicionar efeitos ou animações ao site

Explique UTF-8 e para que usamos.

UTF-8 é uma codificação que permite representar textos em qualquer idioma, sendo a mais utilizada na internet. É compatível com versões anteriores do ASCII e pode representar qualquer caractere Unicode padrão. Essa sigla é usada na linguagem do protuguês Como usamos o UTF-8 É a página de código universal para internacionalização É o padrão para plataformas baseadas em *nix É a codificação de caracteres mais comum da World Wide Web É amplamente utilizada em sistemas operacionais modernos
