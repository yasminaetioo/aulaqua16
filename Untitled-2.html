Explique o que é e o que podemos fazer com o DreamWeaver

Este é o conteúdo da tag Layout P

O Dreamweaver é um programa de design online que permite criar, editar e administrar sites e aplicações online.

Podemos desenvolver muitas coisas dentro desse programa, como: Desenhar, codificar e desenvolver sites Modificar um banco de dados usando comandos ASP Criar e projetar aplicativos baseados na Web Desenvolver as habilidades necessárias para o mercado de trabalho Publicar sites em um servidor externo Realizar pequenas alterações rapidamente e vê-las refletidas imediatamente no servidor
